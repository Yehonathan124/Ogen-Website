# Ogen-Website


שם הפרויקט: 
אתר לאפליקציה

לבד - יהונתן איזנברג 

תיאור הפרויקט: אתר "עוגן"

בניתי אתר שנקרא "עוגן", והמטרה שלו היא לעזור למבוגרים (בני הגיל השלישי) להסתדר יותר טוב ביומיום ולהרגיש בטוחים בבית שלהם.

מה יש באתר?

דף הבית: כאן הצגתי את הרעיון הכללי של האפליקציה. שמתי דגש על זה שהיא נותנת ליווי אישי וסדר יום, ויש שם גם טופס שבו אפשר להשאיר שם וטלפון כדי שנציג יחזור למשתמש לשיחת התאמה.

עמוד פיצ'רים (תכונות): כאן פירטתי מה האפליקציה באמת עושה.
עמוד תמחור: חילקתי את השירות לשלוש רמות מחיר (בסיסי, מתקדם ופרימיום) כדי שכל אחד יוכל לבחור מה שמתאים לו, והוספתי אפשרות להירשם לכל חבילה.

עיצוב וטכנולוגיה: השתמשתי ב-HTML ו-CSS כדי לעצב את האתר. בחרתי בצבעים כחול ולבן כי הם נותנים תחושה של רוגע ומקצועיות. היה לי חשוב שהממשק יהיה נקי וברור כדי שגם אנשים מבוגרים שלא רגילים לטכנולוגיה יוכלו להשתמש בו בקלות.
